## Testing Monitoring

I tested a bunch, but I like [resmoio/kubernetes-event-exporter](https://github.com/resmoio/kubernetes-event-exporter) because it provides exactly what I want and nothing more. Note that there are many "sinks" that it can export to, and I have been just testing with stdout. This means:

```bash
git clone https://github.com/resmoio/kubernetes-event-exporter
cd kubernetes-event-exporter
kubectl create namespace monitoring
# edit deploy/<config> yaml
kubectl apply -f deploy
```

And then you can watch events in the terminal and pipe to file:

```bash
kubectl  logs -n monitoring event-exporter-6bf9c87d4d-v4rtr -f  |& tee ./events-$(date +%s).json
```
